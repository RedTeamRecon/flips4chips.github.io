## Overview

This website is built using HTML5 and CSS3, focusing on a clean and responsive design. The layout is structured using semantic HTML elements, and CSS Flexbox is utilized for layout management. The site includes sections for an "About Me" introduction, a "Projects" showcase, and a "Contact Me" form.

## Features

- **Responsive Design:** Adaptable layout for various screen sizes using media queries.
- **Navigation Menu:** Easy access to different sections of the site.
- **Project Showcase:** A table displaying recent projects with descriptions and thumbnails.
- **Contact Form:** A simple form for visitors to reach out via email.

## Technologies Used

- HTML5
- CSS3
- Flexbox for layout
- Basic form handling (no backend functionality included)
